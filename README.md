# 🌳 Decision Tree Classifier & Regressor (Streamlit App)

An interactive Machine Learning web app built using Streamlit that demonstrates both **classification** and **regression** using the **Decision Tree algorithm** on the Iris dataset.

---

## 🚀 Features

- 🌸 Predict Iris flower type (Classification)
- 🔢 Predict numeric output (Regression)
- 🎛️ Interactive sliders for input features
- ⚙️ Hyperparameter tuning using GridSearchCV
- 📊 Displays model performance metrics

---

## 📂 Dataset

- Iris Dataset (built-in from scikit-learn)
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width

---

## 🧠 Models Used

- DecisionTreeClassifier
- DecisionTreeRegressor
- GridSearchCV for hyperparameter tuning

---

## 📊 Metrics Displayed

### Classification:
- Accuracy Score
- Predicted Class

### Regression:
- Mean Squared Error (MSE)
- R² Score
- Predicted Value
